# Roadmap

## Fase 1: MVP
- frontend navegável;
- condomínios;
- técnicos;
- visitas;
- checklist;
- relatórios;
- contratos;
- dados da empresa;
- persistência local.

## Fase 2: Backend
- API REST;
- banco de dados;
- autenticação;
- uploads persistidos;
- filtros e paginação server-side.

## Fase 3: Contratos
- numeração automática;
- renovação;
- alertas de vencimento;
- PDF oficial;
- assinatura eletrônica.

## Fase 4: Relatórios
- geração real de PDF;
- envio por e-mail;
- versionamento;
- modelos por tipo de serviço.

## Fase 5: Mobile/PWA
- PWA;
- suporte offline parcial;
- câmera integrada;
- sincronização posterior.

## Fase 6: Client Portal
- acesso do condomínio;
- contratos;
- relatórios;
- histórico de visitas;
- abertura de chamados.

## Fase 7: SaaS Scalability
- multitenancy;
- billing;
- observabilidade;
- permissões avançadas;
- alta disponibilidade.
